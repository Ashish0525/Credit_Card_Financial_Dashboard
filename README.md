# Credit_Card_Financial_Dashboard
Power bI Dashboard
Credit Card Data Analysis Report using Power BI
Overview:-
This Power BI dashboard analyzes credit card transaction and customer data to uncover key business insights such as revenue performance, spending patterns, and customer behavior segmentation. The goal was to help financial analysts and decision-makers monitor KPIs effectively through dynamic, interactive visuals.

The client wanted an interactive dashboard to:-
-Track total revenue, amount, and interest earned.
-Understand customer spending patterns by card category and type.
-Compare quarterly and weekly trends in transaction activity.
